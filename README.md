## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/DheerajRay-01/FitPe_assignment.git
   cd healthcare-dashboard
````

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```
